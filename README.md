# Employee Cyberthreat Vigilance System

A comprehensive security awareness platform featuring SSO authentication, interactive training modules, real-time threat reporting, and administrative monitoring.

## Features
- Single Sign-On (SSO) Authentication
- Interactive Training Modules
- Training Progress Tracking
- Real-Time Threat Reporting
- Role-Based Access Control (Admin, Trainer, Employee)
- Secure Multi-Device Access
- Administrative Monitoring Dashboard
- Real-Time Alert & Notifications
- Responsive CSS Styling

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js, Express.js
- Real-Time: Socket.io
- Database: PostgreSQL
- Authentication: OAuth 2.0 / SAML

## Installation
1. Clone the repository
2. Install dependencies: npm install
3. Configure environment variables
4. Run migrations: npm run migrate
5. Start server: npm start
